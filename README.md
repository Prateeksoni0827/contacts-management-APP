# Contacts Management Application

A web-based contact management application built using Angular.

## Features

* View Contacts
* Add New Contact
* Update Contact Details
* Delete Contacts
* Responsive User Interface
* API Integration

## Technologies Used

* Angular
* TypeScript
* HTML
* CSS

## Getting Started

1. Clone repository
2. Run npm install
3. Run ng serve
4. Open http://localhost:4200
